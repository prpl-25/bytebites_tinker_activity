## TF Summary:

The core concept students needed to understand:
Core concept students need to understand is how systems are designed - focusing on the requirements to figure out which attributes to model, and then describing the required parameters and functions to build a system that meets all the requirements.

Where students are most likely to struggle:
Students are most likely going to struggle with the system design portion of the activity. Deciding which classes to model, what attributes are needed, etc.

Where AI was helpful vs misleading:
The AI was misleading while implementing the classes based on the UML diagram. It would forget some parameters and I had to manually enter them and fix mistakes.
It was helpful in planning and implementing the test cases, describing the UML diagrams, and validating the model.

One way they would guide a student without giving the answer:
I would guide them by explaining the Client feature request better. If they understand the requirements and have a good system design plan, implementing it is a lot easier.
